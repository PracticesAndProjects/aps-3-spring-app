Available Routes

User

POST /v1/users

Creates a new user provided the correct payload.

GET /v1/users

Returns all users and their public information.

GET /v1/user/{id}

Returns an user public data by the specified id.

PATCH /v1/user/{id}

Deletes an user by the specified id.

DELETE /v1/user/{id}

Deletes an user by the specified id.

Listing

POST /v1/listings/

Create a new Listing.

GET /v1/listings/

Returns all existing listings.
Optional search parameter. No pagination is implemented.

GET /v1/listings/:id

Returns a specific listing by its id.

DELETE /v1/listings/:id

Deletes an specific listing by its id.
